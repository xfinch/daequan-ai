# Award Criteria

Detailed breakdown of what judges look for at major web design awards.

---

## Awwwards Scoring System

Sites are rated 1-10 across 6 categories. Winners typically score 7.5+ overall.

### 1. Design (Visual Aesthetics)

**What Judges Look For:**
- Originality and creativity
- Visual hierarchy
- Typography mastery
- Color harmony
- Spacing and composition
- Attention to detail

**High Scores Require:**
- Unique visual approach (not template-looking)
- Professional polish
- Consistent visual language
- Appropriate aesthetic for brand

**Common Deductions:**
- Stock-looking templates
- Inconsistent spacing
- Poor color choices
- Typography issues

### 2. Usability (User Experience)

**What Judges Look For:**
- Intuitive navigation
- Clear user flows
- Logical information architecture
- Easy to find key information
- Proper feedback on actions

**High Scores Require:**
- First-time users can navigate easily
- Clear CTAs
- Predictable interactions
- Error prevention
- Helpful error messages

**Common Deductions:**
- Confusing navigation
- Hidden important content
- Broken flows
- No loading states

### 3. Creativity (Innovation)

**What Judges Look For:**
- Novel approaches
- Experimental techniques
- Pushing boundaries
- Unique concepts
- Memorable experiences

**High Scores Require:**
- Something the judges haven't seen
- Risk-taking (that works)
- Conceptual depth
- Artistic merit

**Common Deductions:**
- Derivative work
- Trend-following without innovation
- Gimmicks without purpose

### 4. Content (Quality & Relevance)

**What Judges Look For:**
- High-quality copy
- Relevant imagery
- Clear messaging
- Appropriate tone
- Professional presentation

**High Scores Require:**
- Original content (not placeholder)
- Professional photography/illustration
- Clear value proposition
- Engaging copy

**Common Deductions:**
- Lorem ipsum
- Low-quality images
- Unclear messaging
- Typos/errors

### 5. Mobile (Responsive Design)

**What Judges Look For:**
- Thoughtful mobile adaptation
- Touch-friendly interactions
- Performance on mobile
- Readable typography
- No horizontal scroll

**High Scores Require:**
- Mobile experience as good as desktop
- Optimized images for mobile
- Thumb-friendly navigation
- Appropriate font sizes

**Common Deductions:**
- Desktop-only design
- Tiny touch targets
- Horizontal scrolling
- Poor mobile performance

### 6. Developer (Code Quality)

**What Judges Look For:**
- Clean, semantic HTML
- Modern CSS practices
- Performance optimization
- Accessibility considerations
- Progressive enhancement

**High Scores Require:**
- Valid markup
- Fast load times
- No console errors
- Good Lighthouse scores
- Semantic structure

**Common Deductions:**
- Deprecated practices
- Console errors
- Poor performance
- Invalid HTML/CSS

## Winning Patterns

### What Winners Have in Common:

1. **Polish** - No rough edges anywhere
2. **Originality** - Not derivative of other sites
3. **Purpose** - Every element has a reason
4. **Craft** - Attention to smallest details
5. **Performance** - Fast and smooth
6. **Accessibility** - Works for everyone
7. **Mobile Excellence** - Not an afterthought
8. **Storytelling** - Narrative throughout
9. **Typography** - Type as design hero
10. **Whitespace** - Generous and intentional

## Submission Tips

### For Awwwards:
- Submit when complete (no placeholder content)
- Test thoroughly across devices
- Write clear project description
- Include development credits
- Ensure no broken links

### For CSS Design Awards:
- Choose best day of week (Tue-Thu perform best)
- Have social sharing ready
- Ensure mobile is perfect
- Test all interactions

### For Webby Awards:
- Enter appropriate category
- Provide comprehensive entry
- Include metrics/results
- Professional presentation
- Client testimonials help
