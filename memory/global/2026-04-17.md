# Daily Log — 2026-04-17 (Friday)

## Exports Page — Documentation

### What It Is
The `/comcast/exports` page is a dedicated interface for downloading and copying Comcast visit data for use in external systems (Comcast Business Portal, Excel, other CRMs).

**Location:** `app/comcast/exports/page.tsx`
**URL:** `https://daequanai.com/comcast/exports`

### Why It Exists
Xavier needs to transfer prospect data from the internal Comcast CRM into Comcast's official systems for follow-up and pipeline management. The exports page bridges the gap between:
- **Internal tracking** (MongoDB + SQLite CRM with visit pins, status, notes)
- **External systems** (Comcast Business Portal, Excel sheets, email campaigns)

Without this, Xavier would need to manually query the database or copy data from the map interface — inefficient for bulk follow-up work.

### How It Was Built
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS with existing design system (card, border, accent colors)
- **Data Source:** `/comcast-visits.csv` — a generated CSV file containing all visit records
- **Icons:** Inline SVG components (no external icon library dependency)

**Key Features:**
1. **File Listing:** Displays available CSV exports with metadata (date, location, record count)
2. **Download:** Direct CSV download for Excel/import workflows
3. **"Copy for Comcast" Button:** Parses CSV and formats data as tab-separated values for easy paste into:
   - Comcast Business Portal
   - Excel/Google Sheets
   - Email templates
   - Other CRM systems

**The "Copy for Comcast" Feature:**
- Fetches the CSV file
- Parses headers to find: Business, Contact, Phone, Email, Address, Status, Notes
- Strips quotes and formats as TSV (tab-separated values)
- Copies to clipboard with a single click
- Shows confirmation alert on success

### Expected Behavior

**Normal Flow:**
1. User visits `/comcast/exports`
2. Page fetches `/comcast-visits.csv` and displays file info
3. User clicks either:
   - **"Download CSV"** — saves file locally for import
   - **"Copy for Comcast"** — copies formatted TSV to clipboard
4. User pastes into target system (Comcast portal, Excel, etc.)

**Edge Cases:**
- If CSV doesn't exist: Shows "No export files found" message
- If fetch fails: Logs error, stops loading spinner, shows empty state
- If clipboard copy fails: Shows "Failed to copy data" alert

**Data Format (TSV Output):**
```
Business\tContact\tPhone\tEmail\tAddress\tStatus\tNotes
Business Name\tJohn Doe\t555-1234\tjohn@example.com\t123 Main St\tinterested\tFollow up next week
```
