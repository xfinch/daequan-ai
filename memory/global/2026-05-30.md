# Saturday, May 30, 2026

## Summary

Saturday focused on fixing the Comcast CRM map display issue — visit pins were not showing because addresses lacked geocoded coordinates. Implemented a complete geocoding pipeline to resolve this.

## Activities

### Comcast CRM Map Fix
**Problem:** Map loaded but no visit pins displayed. Investigation revealed 112 of 134 visits had null coordinates.

**Root Cause:** `sync-ghl-to-mongodb.js` set `lat: null, lng: null` with a "Will be geocoded separately" comment, but geocoding was never implemented.

**Solution Built:**
- `scripts/geocode-sqlite.sh` — Shell script to geocode addresses in SQLite using Nominatim API (rate-limited to ~1 address/second)
- `scripts/sync-sqlite-to-mongodb.js` — Syncs geocoded coordinates from SQLite to MongoDB
- `app/api/visits/geocode/route.ts` — API endpoint for future on-demand geocoding

**Status:** Geocoding pipeline functional. Some addresses fail (suite numbers, PO boxes, incomplete addresses) — these need manual cleanup.

### Git Activity
- 2 memory updates throughout the day
- Uncommitted changes: geocoding scripts, API route, SQLite db modifications
