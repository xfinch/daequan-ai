# 2026-02-27 — Comcast Map Fixes

## Summary
Fixed mobile layout and map loading issues for the Comcast territory map at daequanai.com/comcast.

## Issues Fixed

### 1. Map Not Loading ("Loading map..." stuck)
**Problem:** Leaflet + Next.js SSR incompatibility
**Solution:** 
- Created separate `map-component.tsx` with direct Leaflet imports
- Used `next/dynamic` with `ssr: false` in page.tsx
- Added marker icon images to public folder
- Fixed middleware to exclude image files from auth checks

### 2. Mobile Sidebar Too Large
**Problem:** Sidebar took 2/3 of screen on mobile
**Solution:**
- Made sidebar collapsible by default on mobile (50px header visible)
- Added "⚙️ Menu" / "Close" toggle button
- Added floating "🗺️ Show Map" button when expanded
- Shows visit count in collapsed header

### 3. Data Format Mismatch
**Problem:** API returned snake_case, frontend expected camelCase
**Solution:** Updated Flask API (api_server_flask.py) to return camelCase fields
